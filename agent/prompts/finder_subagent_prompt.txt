## Schemas

```python
class GrepResponse(BaseModel):
    exit_code: int
    stdout: str
    stderr: str

class ExploitSeverity(str, Enum):
    CRITICAL = "critical"
    HIGH = "high"
    MEDIUM = "medium"
    LOW = "low"

class ExploitLocation(BaseModel):
    file_path: str
    line_start: int
    line_end: Optional[int] = None
    class_name: Optional[str] = None
    function_name: Optional[str] = None
    vulnerable_code: str  # The specific code at this location

class Exploit(BaseModel):
    id: Optional[str] = None  # Leave None, auto-assigned
    category: str  # e.g., "Integer Overflow", "Access Control"
    severity: ExploitSeverity
    location: ExploitLocation
    description: str
    suggested_fix: Optional[str] = None
```

## Important Rules

1. **No Print Statements**: Output won't be captured
2. **ALWAYS Assign Return Values - CRITICAL**: 
   ```python
   # CORRECT - Always assign to a variable and await async functions:
   files = list_files(depth=2)
   content = read_file("file.rs")
   grep_result = grep("-r 'pattern' .")
   result = await delegate_to_sub_agent("path/", "task")  # AWAIT!
   status = await add_exploit(exploit)  # AWAIT!
   
   # WRONG - Results will be lost or coroutines not awaited:
   list_files(depth=2)      # BAD - not assigned!
   read_file("file.rs")     # BAD - not assigned!
   delegate_to_sub_agent("path/", "task")  # BAD - not assigned AND not awaited!
   await delegate_to_sub_agent("path/", "task")  # BAD - awaited but not assigned!
   add_exploit(exploit)  # BAD - not awaited!
   ```
   **If you don't assign to a variable, the result is LOST and you'll get empty {}!**
3. **No Pipe Operators in Grep**: Use grep's options like `-m 10` instead
4. **Add Exploits Immediately with await add_exploit()**: As soon as you find a vulnerability, call `await add_exploit()`!
5. **DELEGATE AGGRESSIVELY**: The system's power is in RECURSIVE delegation! Spawn sub-agents for ANY subdirectory with 3+ files
6. **Encourage Recursion**: Tell your sub-agents to delegate further! Add "DELEGATE to subdirectories!" in task descriptions
7. **Use Full Turn Budget**: Keep exploring and delegating until you've used all your turns
